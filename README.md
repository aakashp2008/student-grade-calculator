# 🎓 Student Grade Calculator

## 📌 Project Overview

The **Student Grade Calculator** is a Python-based application that calculates a student's grade based on the marks entered by the user. This project demonstrates fundamental Python programming concepts and provides a simple way to evaluate student performance.

---

## ✨ Features

- Accepts student name and marks as input.
- Calculates the total and average marks.
- Determines the grade using conditional statements.
- Displays student information in a clean and readable format.
- Simple, beginner-friendly Python project.
