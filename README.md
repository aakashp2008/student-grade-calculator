# Student Grade Calculator

## About

The Student Grade Calculator is a simple Python project that calculates a student's grade based on the marks entered by the user.

## Features

* Accepts student name and marks
* Calculates grade using conditional statements
* Displays student details and grade
* Easy to understand for Python beginners

## Technologies Used

* Python

## Learning Outcome

This project helped me practice user input, conditional statements, and basic Python programming.
